# Proyecto ESP32 - Puente H

Este proyecto implementa el control de un motor DC usando un puente H con MOSFETs, controlado por un ESP32.

## Componentes
- ESP32
- MOSFET IRFZ44N (canal N)
- Optoacopladores PC817
- Botones de control
- Potenciómetro (PWM)

## Funcionalidad
- Control de dirección (izquierda/derecha)
- Control de velocidad con PWM
- Protección contra inversión brusca
